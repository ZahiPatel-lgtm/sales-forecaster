# Dynamic Sales Forecaster

A machine learning web application built with Python and Streamlit that predicts future sales based on past data. 

# What This App Does
This application allows a user to upload their own historical sales data. It then uses a Machine Learning algorithm to analyze past trends and forecast future revenue. 

# Key Features
* **Dynamic Uploads:** Accepts any standard CSV file with a Date and Sales column.
* **Interactive Data Visualization:** View historical sales trends using Plotly charts.
* **AI Predictions:** Instantly generates a future sales forecast that you can download.

# How to Test the App
If you want to try the app but don't have your own data, you can use the test file provided here!
1. Download the `demo_sales_data.csv` file from this repository.
2. Open the application.
3. Upload the CSV file using the "Browse files" button in the sidebar.
4. Go to the "ML Forecasting" tab and click the **Train Model & Forecast** button!
